# Inbox Folder

This is the drop folder where you can place files for the AI Employee to process.

## How to Use
1. Drop any file into this folder
2. The File System Watcher will detect it
3. AI Employee will process and categorize the file
4. Processed files are moved to appropriate folders

## File Types Supported
- Documents (.pdf, .docx, .txt)
- Spreadsheets (.xlsx, .csv)
- Images (.jpg, .png)
- Code files (.py, .js, .html)

## Processing Workflow
1. File detected by watcher
2. Metadata extracted
3. AI Employee analyzes content
4. Action plan created in Plans folder
5. File moved to appropriate destination
6. Dashboard updated

## Examples
- Drop an invoice → AI processes and logs payment
- Drop a resume → AI categorizes and stores in database
- Drop a meeting agenda → AI schedules and prepares notes
